[solid-contrib](https://github.com/solid-contrib) is an open space to work on software related to [Solid](https://solidproject.org/). This GitHub organisation is administered by the [Open Data Institute](https://theodi.org/) (ODI) - who are the Solid [project stewards](https://theodi.org/insights/projects/odi-and-solid-building-a-future-where-data-works-for-everyone/).

Individual repositories may have a set of owners from the community, if a repository does not have an owner - the ODI will take on ownership responsibility. To ensure transperancy _all_ teams and repositories are public. CODEOWNERS files will be used in each repository to ensure visibility of which individuals or teams are owners of given repositories.

Owners of a repository have full responsibility and autonomy - the ODI will act as if it were an external contributor, the repository owner has final say in which PRs are merged. The only exception is if contents of the repository violate the [Code of Conduct](https://solidproject.org/code_of_conduct) - in which case the ODI Ombuds <solid-ombuds@theodi.org> will moderate content.

If you wish to create a new repository, transfer a repository, or claim ownership of an existing repository create an issue on [solid-contrib/.github](https://github.com/solid-contrib/.github).

The ODI team can be contacted at [solid@theodi.org](mailto:solid@theodi.org).
